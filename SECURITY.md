# Security Policy

## Supported Versions

MarkCraft is a hobby/educational project. Only the `main` branch is actively maintained.

| Version | Supported |
| ------- | --------- |
| `main`  | ✅        |
| older   | ❌        |

## Reporting a Vulnerability

MarkCraft runs entirely in the browser with no server-side component, so the attack surface is narrow. But if you find something that matters — an XSS via a malformed save file, a dependency with a known CVE, supply-chain issues — please report it privately.

**How:**

1. Open a [private security advisory](https://github.com/evan555555555555555/markcraft-clone/security/advisories/new) on this repo (preferred), or
2. Email the repository owner through their GitHub profile.

Please **do not** open a public issue for security problems.

**What to include:**

- A clear description of the issue
- Steps to reproduce (a save file, a URL, a screenshot — whatever shows the bug)
- The version / commit SHA you tested against
- Your assessment of severity

**What to expect:**

- Acknowledgment within 7 days
- A fix or mitigation plan within 30 days for anything confirmed
- Credit in the release notes if you want it

Thanks for helping keep MarkCraft safe.
